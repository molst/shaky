## Artifacts

[on Clojars](https://clojars.org/shaky). If you are using Maven, add the following repository
definition to your `pom.xml`:

``` xml
<repository>
  <id>clojars.org</id>
  <url>http://clojars.org/repo</url>
</repository>
```

With Leiningen:
```
[shaky "0.1"]
```
